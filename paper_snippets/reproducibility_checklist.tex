\subsection{Reproducibility Checklist}
\label{app:reproducibility}

A synthetic-economy paper should be reproducible. The following checklist identifies the components provided in the replication package.

\begin{enumerate}
    \item A script that initializes the synthetic worker population.
    \item A table of occupation-specific parameters.
    \item A table of baseline parameter values.
    \item A random seed used for replication.
    \item A script that generates the worker-period panel.
    \item A script that generates the complexity shock.
    \item A script that computes quality, errors, rework, absorption, fatigue, and productivity.
    \item A script that computes resilience outcomes.
    \item A script that estimates the panel and worker-level regressions.
    \item A script that reproduces all figures and tables.
    \item A README file explaining the order in which scripts should be run.
    \item A synthetic data file sufficient to reproduce all results without requiring proprietary information.
\end{enumerate}
